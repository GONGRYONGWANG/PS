<p>조건을 만족하는 모임 장소를 정하는 것이 가능한지 판별하고, 만약 가능하다면 모든 회원이 모임 장소에 도달하기 위해 이용해야 하는 총 웜홀 비용의 최솟값을 구하시오.</p>
