### 문제 설명

<p>세종이는 문득 자신의 조상님 중에 혈액형이 알맞지 않게 태어난 분이 있는지 궁금해졌다.</p>

<p>세종이네 가족 구성원의 혈액형은 모두 A형, B형, O형, AB형 중 하나다. 혈액형은 다음과 같은 방식으로 결정된다.</p>

<ul>
	<li>모든 가족 구성원은 <strong>유전형</strong>을 갖는다. 유전형은 AA, AO, BB, BO, OO, AB 중 하나의 문자열이다.</li>
	<li>가족 구성원의 혈액형은 유전형에 따라 다음과 같이 결정된다.
	<ul>
		<li>AA, AO: A형</li>
		<li>BB, BO: B형</li>
		<li>OO: O형</li>
		<li>AB: AB형</li>
	</ul>
	</li>
	<li>두 부모 사이에 자식이 생기면, 다음과 같이 자식의 유전형이 결정된다.
	<ul>
		<li>엄마와 아빠의 유전형에서 무작위로 하나씩 문자를 선택한다.</li>
		<li>선택한 두 문자를 사전 순으로 나열한다.</li>
		<li>예를 들어, 엄마의 유전형이 AO고 아빠의 유전형이 BO인 경우, 자식의 유전형으로 가능한 것은 AB, AO, BO, OO이고, 이에 해당하는 혈액형은 각각 AB형, A형, B형, O형이다.</li>
	</ul>
	</li>
	<li>돌연변이 등은 고려하지 않는다.</li>
</ul>
